# How to use tmux

tmuxの使い方をアニメーション豊富で分かりやすく解説するサイトです。

## サイト構成

- **ヒーローセクション** - タイピングアニメーション付きターミナル
- **基本概念** - セッション・ウィンドウ・ペインの階層構造を視覚化
- **ペイン分割デモ** - キーボードをクリックして実際の動作を体験
- **インタラクティブチートシート** - カテゴリ別フィルタリング、コピー機能付き
- **実践シナリオ** - Web開発、サーバー監視、ペアプログラミングの例
- **設定(.tmux.conf)** - おすすめ設定をカテゴリ別に紹介

## ファイル構成

- `index.html` - メインHTML
- `styles.css` - ダークテーマのスタイル
- `script.js` - インタラクティブ機能

## ローカルでの確認

```bash
# 任意の静的ファイルサーバーで起動
npx serve .
# または
python -m http.server 8080
```

## キーボードショートカット

- `?` - チートシートへジャンプ
- `1-5` - 各セクションへジャンプ

## 技術スタック

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter, Fira Code)
